## Local Setup

If you'd like to run my project locally, please read the following steps:

### Frontend Setup

npm install the following depdencies:
-react-router-dom
-axios
-dotenv
-aos --save (animation on scroll)
-firebase

### Backend Setup

npm install the following dependencies:
-express
-pg-promise
-cors
-dotenv
